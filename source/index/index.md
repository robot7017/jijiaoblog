# 一对一及一对二辅导，助力孩子们成长

## 关于我们
我们是一家帮助小学以及初高中同学们实现个性化教育的家教平台，所有的小老师均来自于香港科技大学广州校区。我们致力于为每个孩子提供量身定制的辅导方案，帮助他们提升学习成绩，培养学习兴趣和方法。

## 核心优势

### 🎓 优质师资
- 来自香港科技大学广州校区
- 所有小老师们均成绩优异
- 熟悉各年级教学大纲和考试重点

### 📚 个性化方案
- 能提供全英授课
- 量身定制辅导计划
- 定期跟踪学习进度，调整教学方案

### 📊 透明效果
- 每次课反馈学习情况
- 月度学习报告
- 阶段性成果检测

## 辅导科目

### 小学
- 数学：同步辅导 + 思维拓展
- 英语：口语表达 + 语法基础
- 语文：阅读写作 + 基础知识
- 科学：同步

### 初中
- 数理化：知识点精讲 + 题型归纳
- 英语：中考考点突破
- 语文：阅读理解 + 作文提升

### 高中
- 数理化生：高考冲刺 + 解题技巧
- 英语：听说读写全面提升
- 语文：高考专项训练
